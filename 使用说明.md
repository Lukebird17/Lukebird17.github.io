## 🎯 下一步操作

### 1. 替换头像

将你的照片替换这个文件：
```

```

建议：

- 清晰的个人照片
- JPG 或 PNG 格式

### 2. 修改个人信息

编辑 `hugo.toml`，更新：
```toml
[params]
  author = "你的名字"
  email = "your@email.com"
  location = "你的城市"
  
[params.social]
  github = "https://github.com/你的用户名"
  scholar = "你的 Google Scholar 链接"
  linkedin = "你的 LinkedIn 链接"
  twitter = "你的 Twitter 链接"
```

### 3. 更新页面内容

编辑 `content/_index.md`，修改所有 section 的内容：
- About Me（关于我）
- Publications（发表论文）
- Experience（工作/研究经历）
- Education（教育背景）
- Services（学术服务）
- Miscellaneous（其他信息）

### 4. 部署到 GitHub Pages

#### 方法一：命令行（推荐）

```bash
# 在项目目录下执行
cd /Users/leon/Project/personal-homepage

# 初始化 Git 仓库
git init

# 添加所有文件
git add .

# 提交更改
git commit -m "Initial commit: Personal homepage"

# 设置主分支
git branch -M main

# 关联 GitHub 仓库（替换成你的用户名）
git remote add origin https://github.com/Lukebird17/Lukebird17.github.io.git

# 推送到 GitHub
git push -u origin main
```

#### 方法二：GitHub Desktop

1. 打开 GitHub Desktop
2. 选择 "Add Repository" → "Add Existing Repository"
3. 选择目录：`/Users/leon/Project/personal-homepage`
4. 点击 "Publish repository"
5. 命名为：`Lukebird17.github.io`
6. 确保勾选 "Public"（公开仓库）
7. 点击发布

### 5. 配置 GitHub Pages

1. 打开 GitHub 仓库页面：`https://github.com/Lukebird17/Lukebird17.github.io`
2. 点击 **Settings**（设置）
3. 在左侧菜单找到 **Pages**
4. 在 **Source** 部分选择 **GitHub Actions**
5. 保存后，GitHub 会自动开始构建网站

### 6. 访问你的网站

几分钟后，访问：`https://Lukebird17.github.io`

## ❓ 常见问题

### Q: 网站显示 404
**A:** 
- 确认仓库名是 `Lukebird17.github.io`
- 检查 GitHub Actions 是否成功运行（在仓库的 Actions 标签查看）
- 等待 5-10 分钟后再试

### Q: 图片不显示
**A:**
- 确认图片在 `static/img/` 目录
- 在 markdown 中使用 `/img/文件名.jpg`（以 / 开头）
- 图片文件名不要有中文或特殊字符

### Q: 修改后没有更新
**A:**
- 检查是否成功 push 到 GitHub
- 查看 GitHub Actions 是否运行成功
- 清除浏览器缓存（Ctrl+F5 或 Cmd+Shift+R）

### Q: 样式显示不正常
**A:**
- 清除浏览器缓存
- 检查 `hugo.toml` 中的 `baseURL` 是否正确
- 确认所有文件都已推送到 GitHub

## 🚀 高级自定义

### 修改颜色

编辑 `layouts/index.html`，找到 CSS 变量部分：

```css
:root {
    --primary-color: #01513a;      /* 主色调 */
    --primary-light: #028760;      /* 亮色 */
    --bg-light: #edf9f8;           /* 背景色 */
    --accent-orange: #ff9800;      /* 强调色 */
}
```

### 添加新的导航项

1. 在 `hugo.toml` 添加：
```toml
[[menu.main]]
  identifier = "projects"
  name = "Projects"
  url = "#projects"
  weight = 7
```

2. 在 `content/_index.md` 添加对应 section：
```html
<section id="projects">
<h2>🚀 Projects</h2>
<!-- 内容 -->
</section>
```
